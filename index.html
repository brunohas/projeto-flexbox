O que é Flexbox?

O Flexbox é uma funcionalidade do CSS que permite organizar elementos em uma página da web de forma flexível, podemos organizar elementos em linhas e colunas, alinhar elementos horizontalmente e verticalmente, criar layouts dinâmicos, criar páginas que funcionam bem em diferentes tamanhos de tela, etc. Para usar o flexbox, é necessário especificar um contêiner (pai) e atribuir a propriedade display: flex;, e para os itens (filho) que estão dentro do container, atribuir a propriedade flex: auto; (não permite espaço em branco dentro do elemento pai) para que consiga ficar flexível. Por exemplo: 

                
        <body>
            (pai) → <div class="container">
            (filho)  →  <div class="item">A</div>
            (filho)  →  <div class="item">B</div>
            (filho)  →  <div class="item">C</div>
            (filho)  →  <div class="item">D</div>
            (filho)  →  <div class="item">E</div>
            (filho)  →  <div class="item">F</div>
            </div>
        </body>


        <style>
            div.container {
                background-color: lightsalmon;
                display: flex;
            }
            div.item {
                background-color: lightskyblue;
                border: 1px solid black;
                text-align: center
                flex: auto;
            }
