Ordem dos itens

A propriedade CSS order serve para definir a ordem de elementos em um contêiner flex (se atentar a direção flex-direction). Todo item em flex-box vai ter order: 0;, para alterar acrescentamos o parâmetro order: dentro da tag de estilo. A ordem é crescente, ou seja, do menor número (ou número negativo) para o maior, agora segue um exemplo:

     <div class="container">
         <div class="item" id="a1">A</div>
         <div class="item" id="a2">B</div>
         <div class="item" id="a3">C</div>
         <div class="item" id="a4">D</div>
         <div class="item" id="a5">E</div>
         <div class="item" id="a6">F</div>
     </div>
 </body>

Tamanho dos elementos flexíveis

A propriedade CSS flex-basis define o tamanho inicial dos elementos flex (itens), antes que o espaço restante seja redistribuído. Ela é usada para determinar a largura ou altura mínima de um elemento antes que ele seja redimensionado (parecido com width).

flex-basis: auto; - padrão

Define que o tamanho inicial dos elementos será a largura e altura do próprio elemento, adapta o conteúdo. 


                        
 <body>
     <div class="container">
         <div class="item">texto1</div>
         <div class="item">texto2</div>
         <div class="item">texto3</div>
         <div class="item">texto4</div>
         <div class="item">texto5</div>
         <div class="item">texto6</div>
     </div>
 </body>
 
 <style>
     div.container {
         background-color: lightsalmon;
         display: flex;
         flex-flow: row nowrap;
     }
     div.item {
         background-color: lightslategray;
         border: 1px solid black;
         flex-basis: auto;
     }
 </style>

Expansão

A propriedade CSS flex-grow é usada para definir se os elementos flexíveis se expandem ou não. O valor da propriedade é um número que indica a proporção de espaço em que o item ocupará, se todos os elementos tiverem um flex-grow: 1; o espaço restante será dividido igualmente entre todos. Mas se um elemento tiver o flex-grow: 2; ele vai expandir o dobro do elemento que tem o flex-grow: 1;, e assim adiante. Se o flex-grow: 0; o elemento não irá expandir.

flex-grow: 0; - padrão

Não permite a expansão dos elementos flexíveis.

Redução

A propriedade CSS flex-shrink é usada para definir se os elementos flexíveis se reduzem ou não. Se não houver espaço suficiente na linha, o item flexível será reduzido de acordo com o fator de redução flexível. Se o flex-shrink: 0; o elemento não encolherá, se flex-shrink: 1; os elementos vão encolher, mas não ficará menor que seu conteúdo.

flex-shrink: 1; - padrão

Se a tela diminuir, os elementos também vão diminuir, mas não ficará menor que seu conteúdo.

Também conseguimos mudar o valor, assim mudando a proporção, por exemplo:

O 1° contêiner é ele normal (sem expandir nem reduzir), o 2° contêiner é ele expandido, já o 3° é ele reduzido.

Observe que todos os elementos tem flex-basis: 150px;, porém os flex-grow e flex-shrink são diferentes.

  • O 1° elemento tem um flex-grow: 0; (não permite a expansão do elemento) e flex-shrink: 1; (reduz se a tela for diminuida) = quando aumentar a tela, o elemento não muda, mas quando diminuir a tela, o elemento reduz.
  • O 2° elemento tem um flex-grow: 2; (faz expansão do elemento) e flex-shrink: 0; (não permite a redução do elemento) = quando aumentar a tela, o elemento expande, e quando diminuir a tela, o elemento não reduz.
  • O 3° elemento tem um flex-grow: 1; (não permite a expansão do elemento) e flex-shrink: 2; (reduz se a tela for diminuida) = quando aumentar a tela, o elemento expande, e quando diminuir a tela, o elemento reduz.

Da para perceber que quando expandido o 2° elemento aumentou mais do que o 3° elemento, isso porque, o valor do 2° elemento "fala" que ele vai aumentar 2x mais que o 3° elemento. Isso também ocorre com o 1° elemento e o 3° elemento quando a tela é reduzida. Um valor maior "fala" que o elemento vai ser expandido ou reduzido em dobro, em triplo, etc. Agora vou mostrar outros exemplos (aumente e diminua a tela):
