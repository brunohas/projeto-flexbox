Direção da Flexbox

Para dar a direção no eixo principal (main-axis) em uma flex-box usamos o atributo flex-direction, esse atributo define a direção em que os elementos flexíveis serão exibidos.

flex-direction: row; - padrão

Define a direção do eixo principal (main-axis) em linha, da esquerda para a direita. 

                        
        <body>
            <div class="container">
                <div class="item">A</div>
                <div class="item">B</div>
                <div class="item">C</div>
                <div class="item">D</div>
            </div>
        </body>
        
        <style>
            div.container {
                background-color: lightsalmon;
                display: flex;
                flex-direction: row;
            }
            div.item {
                background-color: darkslategrey;
                border: 1px solid black;   
            }
        </style>

Quebra de linha

Para fazer a quebra de linha de um item flexível usamos o atributo flex-wrap, esse atributo define se os elementos flexíveis devem ser forçados a ficar em uma única linha, ou se podem ser quebrados em várias linhas.

flex-wrap: nowrap; - padrão

Define que não irá ter quebra de linha nos elementos flexíveis. 

                        
        <body>
            <div class="container">
                <div class="item">A</div>
                <div class="item">B</div>
                <div class="item">C</div>
                <div class="item">D</div>
            </div>
        </body>
        
        <style>
            div.container {
                background-color: lightsalmon;
                display: flex;
                flex-wrap: nowrap;
            }
            div.item {
                background-color: darkslategrey;
                border: 1px solid black;
                width: 100px;
            }
        </style>

Define que irá ter quebra de linha nos elementos flexíveis quando um dos elementos não puder ser mais compactado.

Shorthand

A propriedade CSS flex-flow é uma propriedade shorthand (uma mesma declaração inclui vários valores relacionados a mais de uma propriedade) que inclui flex-direction e flex-wrap. Segue todos os tipos de flex-flow:

flex-flow: row nowrap; - padrão

Define a direção do eixo principal em linha, da esquerda para a direita + não permite quebra de linha nos elementos flexíveis. 

                        
        <body>
            <div class="container">
                <div class="item">A</div>
                <div class="item">B</div>
                <div class="item">C</div>
                <div class="item">D</div>
            </div>
        </body>
        
        <style>
            div.container {
                background-color: lightsalmon;
                display: flex;
                flex-flow: row nowrap;
            }
            div.item {
                background-color: darkslategrey;
                border: 1px solid black;
                width: 100px;
            }
        </style>
