Alinhamento eixo principal (main-axis)

Para fazer o alinhamento em relação ao eixo principal (main-axis) em uma flex-box usamos o atributo justify-content. Se atentar a direção em que o eixo está (flex-direction).

justify-content: flex-start; - padrão

Faz o alinhamento no início do eixo principal, grudado no main-start. 

                        
        <body>
            <div class="container">
                <div class="item">A</div>
                <div class="item">B</div>
                <div class="item">C</div>
                <div class="item">D</div>
                <div class="item">E</div>
                <div class="item">F</div>
            </div>
        </body>
        
        <style>
            div.container {
                background-color: lightsalmon;
                display: flex;
                flex-flow: row nowrap;
                justify-content: flex-start;
            }
            div.item {
                background-color: darkslategrey;
                border: 1px solid black;   
            }
        </style>

justify-content: space-between;

Coloca o primeiro elemento grudado no main-start e o último elemento grudado no main-end, todos os outros elementos vão estar no meio com um espaçamento igual entre eles. 

                        
        <body>
            <div class="container">
                <div class="item">A</div>
                <div class="item">B</div>
                <div class="item">C</div>
                <div class="item">D</div>

                <div class="item">F</div>
            </div>

justify-content: evenly;

Parecido com o space-between, porém ele não gruda os elementos no main-start e no main-end, ele faz um espaçamento igual em todos os elementos (espaçamento do main-start até o primeiro elemento, é o mesmo espaçamento entre os elementos, que também é o mesmo espaçamento do último elemento para o main-end). 

                        
        <body>
            <div class="container">
                <div class="item">A</div>
                <div class="item">B</div>
                <div class="item">C</div>
                <div class="item">D</div>
                <div class="item">E</div>
                <div class="item">F</div>
            </div>
        </body>
        
        <style>
            div.container {
                background-color: lightsalmon;
                display: flex;
                flex-flow: row nowrap;
                justify-content: space-evenly;
            }
            div.item {
                background-color: darkslategrey;
                border: 1px solid black;   
            }
        </style>
