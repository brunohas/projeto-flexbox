Alinhamento eixo principal (main-axis)

Para fazer o alinhamento em relação ao eixo principal (main-axis) em uma flex-box usamos o atributo justify-content. Se atentar a direção em que o eixo está (flex-direction).

justify-content: flex-start; - padrão

Faz o alinhamento no início do eixo principal, grudado no main-start. 

                        
    <body>
        <div class="container">
            <div class="item">A</div>
            <div class="item">B</div>
            <div class="item">C</div>
            <div class="item">D</div>
            <div class="item">E</div>
            <div class="item">F</div>
        </div>
    </body>
    
    <style>
        div.container {
            background-color: lightsalmon;
            display: flex;
            flex-flow: row nowrap;
            justify-content: flex-start;
        }
        div.item {
            background-color: darkslategrey;
            border: 1px solid black;   
        }
    </style>

Alinhamento eixo transversal (cross-axis)

Para fazer o alinhamento em relação ao eixo transversal (cross-axis) em uma flex-box usamos o atributo align-items. Se atentar a direção em que o eixo está (flex-direction).

align-items: stretch; - padrão

Estica o elemento para que caiba no contêiner.

↓

→

Organização das linhas

Para fazer a organização das linhas de um contêiner flexível, utilizamos o atributo align-content, só pode ser utilizado quando tiver quebra de linha (flex-wrap: wrap; ou flew-wrap: wrap-reverse;).

align-content: stretch; - padrão

Distribui as linhas uniformemente ao longo do eixo transversal (cross-axis).

↓

→

Alinhamento dos itens

Para fazer o alinhamento de um item individual dentro de uma flexbox, utilizamos o atributo align-self, funciona diretamente no eixo transversal (cross-axis).

- Exemplo dos align-self -

align-self: auto; - padrão

Herda a característica de alinhamento vertical de seu pai, seja ele o align-content ou align-items.

align-self: flex-auto;

Faz o alinhamento grudado o elemento no início do eixo transversal (cross-start).

align-self: flex-end;

Faz o alinhamento grudado o elemento no final do eixo transversal (cross-end).

align-self: center;

Calcula o meio entre o cross-start e o cross-end e deixa os elementos no centro.

align-self: stretch;

Estica o elemento para que caiba no espaço inteiro do contêiner. 


                        
    <body>
        <div class="container">
            <div class="item" id="a1">auto</div>
            <div class="item" id="a2">start</div>
            <div class="item" id="a3">end</div>
            <div class="item" id="a4">center</div>
            <div class="item" id="a5">stretch</div>
        </div>
    </body>
    
    <style>
        div.container {
            background-color: lightsalmon;
            height: 100px;
            display: flex;
            flex-flow: row nowrap;
            align-items: center;
        }
        div.item {
            background-color: darkslategrey;
            border: 1px solid black;
            width: 200px;
            text-align: center;   
        }
        div#a1 {
            align-self: auto;
        } 
        div#a2 {
            align-self: flex-start;
        }
        div#a3 {
            align-self: flex-end;
        }
        div#a4 {
            align-self: center;
        }
        div#a5 {
            align-self: stretch;
        }
    </style>
